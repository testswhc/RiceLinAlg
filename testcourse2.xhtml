The previous problem gave us an example of a system that is consistent, but has more than one solution. The augmented matrix

How many types of elementary row operations are there?

Answer:

  • Switch two rows.

  • Multiply a row by a non-zero scalar.

  • Add a scalar multiple of one row to another row. (More precisely, replace one row with the sum of itself and a scalar multiple of a different row.)

There are three types of elementary row operations that one can apply to a matrix.

  • Switch two rows.

  • Multiply a row by a non-zero scalar.

  • Add a scalar multiple of one row to another row. (More precisely, replace one row with the sum of itself and a scalar multiple of a different row.)

If one can obtain augmented matrix [mathjaxinline]B[/mathjaxinline] from augmented matrix [mathjaxinline]A[/mathjaxinline] via a series of elementary row operations, we say that [mathjaxinline]A[/mathjaxinline] and [mathjaxinline]B[/mathjaxinline] are row-equivalent.

If [mathjaxinline]A[/mathjaxinline] and [mathjaxinline]B[/mathjaxinline] are row-equivalent, the system of linear equations represented by [mathjaxinline]A[/mathjaxinline] and the system of linear equations represented by [mathjaxinline]B[/mathjaxinline] have the same set of solutions.
